# Webinar Posters

Place your webinar poster images in this folder.

## Required Images:

1. **session-1-poster.jpg** - Poster for "Brillianton Achievers' Dialogue Session 1" (December 27, 2025)
   - This is the past webinar poster

2. **session-2-poster.jpg** - Poster for "Unlock Your Path to Oxford, Manchester & Chevening!" (January 24, 2026)

## Image Requirements:

- Format: JPG, PNG, or WebP
- Recommended size: 1200x800px or similar aspect ratio
- File names must match exactly: `session-1-poster.jpg` and `session-2-poster.jpg`

After adding the images, they will automatically appear on the webinars page.
